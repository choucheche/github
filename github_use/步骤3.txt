新建分支

git branch demo



切换到分支

git checkout demo



对代码进行修改

git add .

git commit -m '1'

git push origin demo


上传demo分支的代码到demo


git checkout master


切换到master分支


git fetch origin demo


从demo分支上下载代码

git merge demo


在本地master上合并demo代码


git pull origin master

同时用了fetch 和 merge



git push origin master

上传master的代码



git branch -d demo
删除本地分支

git branch -D demo
强制删除本地分支


git push origin :demo
